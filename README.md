# sistema-nomina-python
Aplicación en Python para calcular la nómina de empleados según días trabajados y cargo laboral. Proyecto académico UNAD.
Sistema de Nómina en Python

Aplicación desarrollada en Python con interfaz gráfica utilizando Tkinter.
Permite registrar empleados y calcular el pago de nómina según los días trabajados y el cargo laboral.
